华夏历 Huaxia Calendar — Universal Planetary Formula

Not a calendar. A methodology that generates calendars.

The traditional Chinese lunisolar calendar encodes a parametric timekeeping structure: Cp = ΦA(Θ₁, {Ψ∈A}) ⊕ ΦB(Θ₁…Θm, {Ψ∈B}). Solar terms divide the orbit by ecliptic longitude (N equal angles, Kepler-compatible by construction); a satellite with synodic period Tᵢ in the Mode A interval [Y₁/N, 2Y₁/N) drives intercalation automatically via the no-Zhongqi rule. The Gregorian calendar is the special case with Earth's parameters hardcoded.

  • Cross-system verification: 10 real systems (Earth, Mars, Jupiter, Venus, TRAPPIST-1 e/f, Proxima b, Kepler-16b, TOI-5624 e, b Cen b, GJ 536 c) — all produce defined calendars; Gregorian works for exactly one.
  • Exomoon hunter: Kepler-1625 b I (synodic ≈ 20.35 d, Tᵢ/Z = 85%) would be the second Mode A instance — the candidate itself remains observationally disputed.
  • Verification: independent Python scripts recompute all tables; solar-term timing cross-checked against the JPL DE421 ephemeris (max Δ ≈ 14 min over 2 years, CI-enforced < 30 min; calendar decisions are day-granular).
